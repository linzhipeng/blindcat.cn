## 获取文章列表

**URL：** 
- ` /article `
  
**请求方式：**
- GET 

**备注：**
- 免登录

**参数：** 

|参数名|必选|类型|说明|默认值|
|:----    |:---|:----- |-----   |------  |
|pageNum |否  |int |文章页码   |1 |
|articleNum |否  |int | 每页文章数    |10 |

 **数据模型**

``` 
{
    "state": 请求状态,
    "data": {
        "articleCount": 符合查询的总文章数,
        "pageCount":  符合查询的总页数,
        "searchType": 搜索类型,
        "pageNum": 搜索页码,
        "listData": [
            {
                "_id": 文章id,
                "title": 文章标题,
                'preview': 文章预览图,
                "writer": {
                    "_id": 作者id,
                    "username": 作者昵称,
                    "account": {
                        "email": 作者邮箱
                    }
                },
                "abstract": 文章正文,
                "likeNum": 点赞数,
                "aticleCollect": 收藏数,
                "readTimes": 阅读数,
                "publishTime": 文章发布时间,
                "tags": 文章标签数组
            }
        ]
    }
}
```
